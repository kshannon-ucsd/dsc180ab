Sepsis - Using Clinical Healthcare Data Science to Identify and Combat an Infectious Killer

Data science capstone domain (DSC 180AB {{ site.data.variables.section }})

Fall & Winter {{ site.data.variables.year }} - Prof. Kyle Shannon

Schedule

Week Topic
1 Start with a thorough reading of the paper. Understand the problem statement, methods used, and the results obtained. Also, start reading about the MIMIC-III dataset and understand its structure and the kind of data it contains.
2 Continue studying the MIMIC-III dataset. Apply for access to the MIMIC-III dataset, as this process can take some time. Start learning about the domain of sepsis and critical care.
3 Once access to the MIMIC-III dataset is granted, start exploring the dataset. Understand the different tables and the relationships between them. Also, continue reading about sepsis and critical care.
4 Begin data preprocessing. This includes cleaning the data, handling missing values, and extracting the relevant features needed for the study. Also, start reading about the methods used in the paper, such as k-means clustering and Latent Class Analysis (LCA).
5 Continue with data preprocessing. Start learning about the Elixhauser comorbidity index and how it is used to categorize diseases. Also, continue reading about k-means clustering and LCA.
6 Start implementing the k-means clustering algorithm on the preprocessed data. Also, continue learning about LCA.
7 Continue with the implementation of k-means clustering. Start implementing the LCA on the data.
8 Complete the implementation of LCA. Start analyzing the results obtained from k-means clustering and LCA.
9 Implement the network analysis on the data. Visualize the relationships between morbidities using the appropriate tools.
10 Continue analyzing the results. Start learning about network analysis and how to visualize the relationships between morbidities.
